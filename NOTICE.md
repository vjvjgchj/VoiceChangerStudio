# Third-Party Notices

VoiceChangerStudio keeps the upstream voice changer runtime code needed for local inference. The main source license is listed in `LICENSE`.

Some runtime model families and optional pretrained assets have their own notices:

- Diffusion SVC and DDSP SVC may use DiffSinger Community Vocoders. Check the vocoder license before redistributing pretrained vocoder files: https://openvpi.github.io/vocoders/
- Beatrice JVS Corpus Edition has its own upstream license notes. Check the upstream Beatrice runtime directory before redistributing bundled Beatrice model assets.

Do not publish private voice models, pretrained weights, recordings, logs, or local Python environments unless you have verified the redistribution rights for those files.
